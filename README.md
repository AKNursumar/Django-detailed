Django Detailed 🚀

A beginner-friendly yet detailed Django learning repository created by AK Nursumar while learning backend web development using the Django framework.
This repository contains practical implementations of core Django concepts including routing, views, templates, forms, static files, models, authentication, and more.


📌 About The Project

This project is part of my Django learning journey during my BTech IT studies at CHARUSAT.
The goal of this repository is to understand Django deeply by building features step-by-step instead of only following theory.

It includes:

Django project setup
URL routing
Views & Templates
Template inheritance
Static & media files
Forms handling (GET/POST)
User authentication
Database models
Admin panel usage
Migrations
CRUD basics
And more...
